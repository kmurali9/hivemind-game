Firebase Setup Needed

To play with friends, you need to connect a (free) Firebase project. It takes about 3 minutes.

Quick steps:

1. Go to console.firebase.google.com
2. Click "Add project" and give it a name
3. Go to Project Settings → General → scroll to "Your apps" → click the web icon (</>)
4. Register the app (any name) and copy the config object
5. Paste the values into the FIREBASE_CONFIG at the top of this file
6. In the Firebase console, go to Authentication → Sign-in method → Enable "Anonymous"
7. Go to Firestore Database → Create database → Start in test mode
8. Refresh this page!
